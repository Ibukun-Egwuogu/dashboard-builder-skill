Why does an API URL look like gibberish?
When you open a URL like https://api.frankfurter.app/latest in your browser, you see raw JSON — data in its machine-readable form. It's not broken. It's just not designed for human eyes. Think of it like the back of a restaurant kitchen: everything you need is there, but it needs a plate before it looks like food.

Below you can see each API's live data rendered cleanly as a table. The How to connect data tab shows you exactly how to plug these URLs into Power BI, Tableau, or Python so your dashboard updates automatically.

An API URL is just a live data pipe. You paste it into your BI tool and it pulls the data in as a table — automatically, on a schedule you control. Here's how to do it in each major tool.

📊
Power BI Desktop
Works with any JSON or CSV API — no plugin needed
1
Open Power BI Desktop → Home → Get Data → Web
2
Paste your API URL, e.g. https://api.frankfurter.app/latest?from=USD&to=NGN,GBP,EUR → click OK
3
Power Query opens. Click into the rates record to expand it — you'll see NGN, GBP, EUR as columns.
4
Click Into Table → expand columns → rename them → Close & Apply. Your live rates are now in your report.
5
To schedule refresh: Publish to Power BI Service → open your dataset → Settings → Scheduled refresh → set to Daily.
Tip: For the World Bank GDP API, add ?format=json&mrv=10&per_page=10 to get the last 10 years. Power Query will expand the nested JSON automatically.
📈
Tableau Desktop
Best with a Google Sheets middle layer for beginners
1
For static snapshots: open the API URL in your browser → save the page as a .json file → connect via Connect → To a File → JSON
2
For live data (easiest route): use Google Sheets as a middle layer — see the Google Sheets section below.
3
Connect Tableau to your Google Sheet via Connect → To a Server → Google Sheets. The Sheet auto-refreshes and Tableau picks up new data on each publish.
📋
Google Sheets → Looker Studio
The most beginner-friendly live data setup
1
Open a new Google Sheet → in cell A1 type: =IMPORTDATA("https://api.frankfurter.app/latest?from=USD"). The exchange rates load automatically.
2
Add column headers, clean up the data. Google Sheets auto-refreshes the IMPORTDATA formula every time the sheet is opened.
3
Open Looker Studio → Add data → Google Sheets → select your sheet. Done — your report now has live data.
4
For JSON APIs (like World Bank), use Extensions → Apps Script to write a small script that fetches the JSON and writes it to the sheet on a daily timer.
🐍
Python → Power BI / Tableau
Most flexible — great for portfolio projects
1
Install the libraries: pip install requests pandas openpyxl
2
Run the script below to pull the API and save as a CSV file.
3
In Power BI: Get Data → Text/CSV → point to your saved file. Every time you re-run the script, Power BI picks up the new data on the next refresh.
# Exchange rates → CSV import requests, pandas as pd r = requests.get("https://api.frankfurter.app/latest?from=USD") data = r.json() df = pd.DataFrame(data['rates'].items(), columns=['Currency', 'Rate']) df['Date'] = data['date'] df.to_csv("exchange_rates.csv", index=False) # Nigeria GDP → CSV r2 = requests.get("https://api.worldbank.org/v2/country/NG/indicator/NY.GDP.MKTP.CD?format=json&mrv=15&per_page=15") gdp = r2.json()[1] df2 = pd.DataFrame([{'Year': x['date'], 'GDP_USD': x['value']} for x in gdp if x['value']]) df2.to_csv("nigeria_gdp.csv", index=False)
Automate it: Schedule the script to run daily using Windows Task Scheduler (Windows) or cron (Mac/Linux). Your CSV updates every morning and Power BI refreshes from it automatically.
